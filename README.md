## Installation

1. Clonez ce dépôt :

```plaintext
git clone https://github.com/smail7777/todo.git
cd todo
```


2. Installez les dépendances :

```plaintext
npm install
```


3. Démarrez l'application :

```plaintext
node app.js
```


4. Ouvrez votre navigateur et accédez à :

```plaintext
http://localhost:3000
```
